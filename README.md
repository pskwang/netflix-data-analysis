# 🎬 Netflix 데이터 분석 포트폴리오

## 📌 프로젝트 개요
넷플릭스 콘텐츠 데이터를 MySQL과 Python으로 분석한 포트폴리오입니다.

## 🛠️ 사용 기술
- **Database**: MySQL
- **Language**: Python
- **Library**: Pandas, Matplotlib, SQLAlchemy

## 📊 분석 내용
1. Movie vs TV Show 비율
2. 연도별 콘텐츠 추가 추이
3. 국가별 콘텐츠 TOP 10
4. 시청 등급 분포
5. 장르 TOP 10
6. 감독별 작품 수 TOP 10

## 📈 대시보드
![Netflix Dashboard](netflix_dashboard.png)

## 💡 핵심 인사이트
- Movie가 전체의 69.6%로 TV Show보다 압도적으로 많음
- 2019년에 콘텐츠 추가가 최고조에 달함
- 미국이 전체 콘텐츠의 약 32% 차지
- TV-MA 등급이 36.4%로 성인 콘텐츠가 가장 많음

## 📂 파일 구조
- `main.py` - Python 분석 및 시각화 코드
- `analysis_queries.sql` - MySQL 분석 쿼리
- `netflix_dashboard.png` - 시각화 결과
